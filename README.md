# 連接氣象局OpenData

## 使用Java Swing技術，以物件導向的程式特性建構平易近人的天氣查詢工具。

## 設計功能
  ● 各個縣市天氣查詢、攝氏華氏溫度之轉換，以及當前時間顯示。
  ● 使用javax.imageio讀取圖片，並使用JSlider來實現圖片放大縮小。
